📎 店铺信息抓取工具

在抖音/美团店铺页面点一下，自动提取信息填入管理后台

安装方法（30秒）

  1. 把下面这个橙色按钮拖到浏览器书签栏
🖱 拖我到书签栏 → 抓取店铺信息

如果书签栏没显示：Chrome 按 Ctrl+Shift+B，Edge 按 Ctrl+Shift+B

⬇
  1. 打开抖音或美团的店铺页面
  2. 点击书签栏里的 「抓取店铺信息」
  3. 自动跳转到管理后台，信息已帮你填好

支持抓取的平台

🎵 抖音店铺 🛵 美团店铺

在以上任意平台的店铺主页或团购评价页面点击书签即可。
自动提取：店铺名称、评分、人均价格、页面链接。

测试：粘贴任意文字试试提取效果

模拟从店铺页面抓取的文字，看看能提取出什么

它会抓取什么

📌 页面标题 → 自动填入店铺名称
⭐ 评分数字 → 自动填入评分（如 4.8分）
💰 人均价格 → 自动填入价格（如 人均¥120）
🔗 页面链接 → 自动填入评价跳转链接
